Axon Basket Sample

POST to /baskets to create a basket. POST to /baskets/{id} to add a line item using the following JSON.

curl -X POST \ http://localhost:8080/baskets \ -H 'cache-control: no-cache' \ -H 'content-type: application/json'

curl -X POST \ http://localhost:8080/baskets/{basketId} \ -H 'content-type: application/json' \ -d '{ "productId":"test-product-id", "productName":"A Name", "priceInCent": 42, "quantity": 23 }'

GET /baskets to list all available baskets.
